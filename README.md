# Projeto de Machine Learning – Análise e Previsão de Dados

## Descrição

Este repositório contém as atividades desenvolvidas para a disciplina de **Machine Learning**, do curso de **Análise e Desenvolvimento de Sistemas**.  
O objetivo do trabalho foi aplicar conceitos fundamentais de aprendizado de máquina na exploração, tratamento e modelagem de dados, utilizando Python e bibliotecas voltadas à análise preditiva.

## Conteúdo do Repositório

O projeto está dividido em duas etapas principais:

### 1. Exploração e Preparação dos Dados

- Carregamento e verificação do conjunto de dados.
- Análise estatística e identificação de possíveis correlações.
- Tratamento de valores ausentes e normalização das variáveis.
- Visualização de distribuições e relações entre atributos.

### 2. Modelagem e Avaliação

- Criação e treinamento de modelos de regressão.
- Avaliação de desempenho utilizando métricas como **RMSE** e **R²**.
- Interpretação dos resultados obtidos a partir dos gráficos e métricas.
- Ajustes nos hiperparâmetros e comparação entre diferentes modelos.

## Ferramentas e Tecnologias

- **Linguagem:** Python
- **Bibliotecas principais:** Pandas, NumPy, Scikit-learn, Matplotlib
- **Ambiente de desenvolvimento:** Jupyter Notebook / VS Code
